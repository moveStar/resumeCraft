## 项目简介

基于`tdesign-vue-next` 组件库的简历生成工具，通过用户输入信息，自动生成简历。

## 开发
````bash
node 22.18.0
vue 3.4.21
typescript 5.4.4
vite 5.2.8
sass 1.91.0
````

### 安装依赖

```bash
pnpm install
```

### 启动项目

```bash
pnpm run dev
```

## 构建

### 构建正式环境

```bash
npm run build
```
